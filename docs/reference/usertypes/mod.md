# Script

### `script.name`
  > Returns `string`
  
  > The name of the script
  
### `script.version`
  > Returns `string`
  
  > The version of the script
  
### `script.api_version`
  > Returns `string`
  
  > The api version of the script
  
### `script.description`
  > Returns `string`
  
  > The description of the script
  
### `script.authors`
  > Returns `sequence<string>`
  
  > The authors of the script
